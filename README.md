# Allthing Decode — Luxury Interior Design Kerala

Elevating spaces beyond the ordinary — where luxury meets purposeful design. Creating environments that inspire progress and sophistication.

## Project info

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```
